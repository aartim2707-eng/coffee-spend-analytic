# coffee-spend-analytic
Power BI dashboard analyzing coffee sales trends, spending patterns, and customer behavior. Insights include peak hours, weekday vs weekend spend, and time-of-day preferences. Built with Excel + Power BI, this project helps coffee shops optimize operations and boost sales.

# ☕ Coffee Trends & Spend Analysis Dashboard

## 📌 Overview
This project provides a **Power BI dashboard** that analyzes coffee sales trends, customer spending behavior, and peak consumption hours.  
The dashboard is built using the `Coffee_sales` dataset, which includes attributes such as:
- Transaction hour
- Payment type
- Amount
- Coffee name
- Time of day
- Weekday
- Month
- Date

The goal is to help businesses understand **consumer behavior patterns** and optimize sales strategies.

---

## ✨ Features
- **Total Spend Analysis**: Track overall revenue and average spend.
- **Peak Hours Identification**: Discover the busiest times of the day.
- **Weekday vs Weekend Trends**: Compare spending behavior across weekdays and weekends.
- **Coffee Type Popularity**: Visualize which coffee types drive the most sales.
- **Time of Day Insights**: Morning, afternoon, and evening consumption patterns.
- **Monthly Spend Trends**: Seasonal variations in coffee sales.
